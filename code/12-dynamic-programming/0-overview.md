# Dynamic Programming

## Complexity Comparison
Brute Force     Memoization(Top Down)  Bottom-Up     Optimized
Time: O(2^n)      Time: O(n)              Time: O(n)     Time: O(n)
Space: O(n)       Space: O(n)             Space: O(n)     Space: O(1)

## Top Down vs Bottom Up
Top Down:
- Start from the top and work down to the bottom
- Use recursion to solve the problem

Bottom Up:
- Start from the bottom and work up to the top
- Use iteration to solve the problem
