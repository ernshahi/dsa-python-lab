# System Desing Inteview vs Product Design Interview
- Product Design Interview:
  - Focuses bit more on the design of APIs, datamodels, and desing a user facing application.
  - Involves designing features, user interfaces, and user experiences.
- System Design Interview:
  - Focuses on the architecture of large-scale systems, scalability, and performance.
  - Involves designing backend systems, databases, and APIs.
